
Imagine - Image/Animation Viewer for Windows


*** Features ***

- Views Image/Animation files with the fastest speed.
  (NO depends on their extensions)
- Numerous image formats supported.
- Shows informations of image/animation files.
- Shows EXIF(JPEG) informations.
- Shows transparency.
  (Imagine also shows the alpha channel in 32bit image.)
- Extracts a frame(or whole frame) from Animation file.
- Exports Animation file to Animated GIF file/AVI/Flic/Pro Motion Animation.
- Manipulates image
  (Flip,Grayscale,Negative,Change color depth,Swap colors,Rotate,Resize
   Effect filter,Edit palette,Transparency set)
- Tiny & Fast & Pure & Neat executable.
- Fully customizable. (Keyboard shortcut, Toolbar)
- Plugin add-on available.
- Multilingual support
- NO depends on any OLE method or any other external viewer.
- NO creates or uses any registry value.
  (except for only when setting desktop wallpaper)
- Shell extension support.
  (Context menu in Windows Explorer is available.)

*** Installation (Only in Total Commander) ***

1. Unzip all files to any directory
2. Choose Configuration - Options
3. Open the 'Edit/View' page
4. Click 'Configure internal viewer'
5. Click 'LS-Plugins' button
6. Click 'Add' button and select the Imagine.wlx
7. Click 'OK'


*** Default Keyboard shortcuts (Customizable) ***

Ctrl+O - Open image
Ctrl+S - Save image
Space - Next image
Backspace - Previous image
Ctrl+C - Copy to clipboard
I - Image information
V - Vertical flip
H - Horizontal flip
Shift+G - Grayscale
Shift+T - Negative
Ctrl+D - Change color depth
L - Rotate left
R - Rotate right
Shift+R - Rotate free
Ctrl+Shift+E - Set as wallpaper (Centered)
Ctrl+Shift+T - Set as wallpaper (Tiled)
Ctrl+Shift+S - Set as wallpaper (Stretched)
Ctrl+I - Filter browse
E - Palette edit
Shift+S - Status bar
B - Fit only big images
Enter - Full screen
T - Show transparency
+ - Zoom in
- - Zoom out
* - Zoom to 100%
Alt+Enter - Preferences
Delete - Delete current file (to Recycle Bin if available)
Shift+Delete - Delete current file (delete directly)

(Animation shortcuts)
Pause - Pause
Ctrl+Pause - Next frame
S - Show frame number
Ctrl+E - Extract all frames

(Multiple page shortcuts)
Shift+PageUp - Previous page
Shift+PageDn - Next page
Shift+Home - First page
Shift+End - Last page
